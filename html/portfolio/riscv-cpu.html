3-Stage RISC-V CPU

For this project, my teammate and I designed and optimized a 3-stage pipelined RISC-V CPU capable of executing the complete RV32I Base Instruction Set. Features include (but are not limited to) handling of control and data hazards; interfacing with the CPU through memory-mapped I/O to send and receive bytes to and from the UART; and branch prediction.

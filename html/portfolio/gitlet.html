Gitlet

This project is a mini version control system modeled after Git, intended to help developers collaborate and backup their work. Written in Java, users can save the contents of files in the form of "commits." Gitlet maintains a history of them, such that one may revert and merge changes to their current file by simply referencing a particular commit.
